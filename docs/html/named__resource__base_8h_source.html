13 // limit maximum resource name length to this
14 // Windows limit is 260, POSIX limit is 251, though OSX limit seems to be 30 :S
15 // Windows names cannot contain a backslash '\'
16 // POSIX names must begin with a / for portability if used between

18 // and cannot contain a / anywhere else.
